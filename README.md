# Real-Time Face Detection Using OpenCV

A simple real-time face detection project built using Python and OpenCV.

This application uses a webcam to detect human faces and draws rectangles around detected faces in real-time.

---

# 🚀 Features

✅ Real-time webcam face detection  
✅ Detects multiple faces  
✅ Live video processing  
✅ Rectangle overlay on faces  
✅ Beginner-friendly OpenCV project  

---

# 🛠 Technologies Used

- Python
- OpenCV
